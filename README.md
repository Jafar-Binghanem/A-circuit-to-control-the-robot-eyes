# A-circuit-to-control-the-robot-eyes
An arduino-circuit to control the robot eyes. It uses PWM to reduce the energy losses in the LEDs  

**Task 3**    
  
   **A. The code**  

```ruby
// C++ code
//
void setup()
{
  pinMode(5, OUTPUT);
   pinMode(6, OUTPUT);
}

void loop()
{
  analogWrite(5,30);
  analogWrite(6,30);
 
}
  ```
